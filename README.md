# Network-Intrusion-and-Traffic-Dashboard
An AI-engineered Python dashboard analyzing 25,192 network sessions to isolate a 46.6% threat ratio (11,743 attacks) and 14 root compromises (p. 1). Uses log scaling to map payloads across TCP/UDP/ICMP (p. 1). Visualizes intrusion states (14,973 SF flags) and reveals private traffic as the top vulnerability vector at a 92.1% malicious focus rating.
